# Rachit Agrawal — Personal Portfolio

A personal website exploring the intersection of computer science, game theory,
research, and the humanities.

**Live site:** https://rachitagrawal146.github.io/

## Structure
- `index.html` — Main portfolio (About, Timeline, Projects, Theatre, Achievements, Contact)
- `projects.html` — Detailed write-ups of the Game Algorithms Engine projects
- `images/` — Portraits and theatre/trek photographs

## Related
- [Game Algorithms Engine](https://github.com/RachitAgrawal146/game-algorithms-java) — interactive research project exploring minimax, constraint satisfaction, and adversarial search

## Stack
Pure HTML + Tailwind CSS (CDN) + vanilla JavaScript. No framework, no build step.
